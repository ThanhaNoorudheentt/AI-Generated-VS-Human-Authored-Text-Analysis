# AI-Generated-VS-Human-Authored-Text-Analysis

## Project Overview
With the rise of LLMs (GPT-4, Claude, Gemini), distinguishing machine-generated text from human writing is a critical challenge for academic integrity and misinformation detection. This project implements a robust NLP pipeline to classify text origins using both traditional machine learning and deep learning architectures.

## Objectives
- **Build a reliable classifier** using Logistic Regression and DistilRoBERTa.
- **Analyze linguistic signatures** that differentiate AI (formal/structured) from Human (informal/varied).
- **Test Robustness** against "Humanization" attacks (slang and typos).
- **Evaluate Ethics** regarding false positives in formal human writing.

## Dataset Details: HC3
The project utilizes the **Human ChatGPT Comparison Corpus (HC3)**, a multi-domain dataset designed for the study of AI-generated text detection.

- **Source:** [Hello-SimpleAI/hc3](https://huggingface.co/datasets/Hello-SimpleAI/hc3)
- **Total Samples:** ~170,898 rows (after flattening)
- **Composition:**
  - **Human Authored (Label 0):** Responses curated from human experts and platforms like Reddit, Wikipedia, and WikiHow.
  - **AI Generated (Label 1):** Corresponding answers generated by ChatGPT (GPT-3.5) using identical prompts.
- **Domains Covered:** 
  - Open-domain (ELI5, Reddit)
  - Computer Science & WikiHow
  - Legal, Medical, and Financial questions
- **Preprocessing:** 
  - Flattened nested answer lists into individual rows.
  - Removed "AI-fingerprint" prefixes (e.g., "As an AI language model...").
  - Filtered out short responses (<10 words) to ensure linguistic density.

## Methodology & Models
1. **Data:** HC3 (Human ChatGPT Comparison Corpus) - 170k+ rows of flattened text.
2. **Baseline:** Logistic Regression with TF-IDF Vectorization (Accuracy: ~97%).
3. **Advanced Model:** Fine-tuned `distilroberta-base` (Accuracy: 98.6%).
4. **Adversarial Testing:** Manual "Style-Transfer" attacks to test model vulnerability.

## Key Findings
- **AI Signature:** High frequency of "hedging" (may, might) and formal transitions (however, overall).
- **Human Signature:** High use of personal pronouns (my, I) and informal fillers (basically, pretty).
- **Vulnerability:** The model was successfully fooled by adding informal slang, shifting an AI classification (99% confidence) to Human (99% confidence) with minimal content changes.

## Conclusion

This project demonstrates that while transformer-based models like RoBERTa can distinguish between AI-generated and human-authored text with high accuracy (98.6%), they remain highly susceptible to 'Style-Transfer' attacks. Our analysis revealed that AI models have a distinct linguistic signature characterized by formal transition words (e.g., 'however', 'overall') and a lack of personal pronouns. However, by intentionally introducing 'human' artifacts such as slang, typos, and informal syntax, we successfully shifted the model’s confidence from 99% AI to 99% Human. This highlights a critical challenge for 2026 and beyond: as LLMs become more adept at mimicking human idiosyncrasies, detection based on stylistic patterns will become increasingly unreliable, potentially leading to false positives for formal human writers or non-native English speakers.

Detection is a "cat-and-mouse" game. While deep learning provides high accuracy on standard datasets, it is not foolproof against intentional stylistic manipulation. Future work should focus on cross-lingual detection and better generalization across different LLM families (Claude, Gemini).
